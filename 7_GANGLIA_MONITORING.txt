On Management_node:
	Install meta daemon, monitoring daemon and web UI by executing:
		yum install ganglia ganglia-gmetad ganglia-web ganglia-gmond
	Management Node Configuration: 
		vi /etc/ganglia/gmetad.conf
	Edit the data_source line so that it looks like this:
		data_source  “my_cluster"    192.168.0.1

On Client_node:
	Install monitoring daemon by:
		yum install ganglia ganglia-gmond

All_Nodes
	Configuration:
		vi /etc/ganglia/gmond.conf
	Edit so that the file has these configs
		cluster {
  			name = “my_cluster"
  			}
		udp_send_channel {
  			#mcast_join = 239.2.11.71
  			host = 192.168.0.1
  			}
		udp_recv_channel {
  			#mcast_join = 239.2.11.71
  			#bind = 239.2.11.71
			}
		tcp_accept_channel {
  			port = 8649
  			}
Start the services on server node:
	Restart httpd:
		systemctl start httpd
      	systemctl enable httpd
	Start gmond:
		systemctl start gmond
      	systemctl enable gmond
	Start gmetad:
		systemctl start gmetad
      	systemctl enable gmetad
   		vi /etc/httpd/conf.d/ganglia.conf
   			Comment all lines using # and uncomment last line = Require all granted
Start the services on client node:
	Start gmond:
		systemctl start gmond
       	systemctl enable gmond
Access from the Webbrowser using http://192.168.0.1/ganglia
